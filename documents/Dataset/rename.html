rename

rename(newName)

Fast rename dataset, also change datasetKey, datasetLabel to new name

Parameters

Required:
  • newName:String. New name of dataset
Optional:
None

Returns:

  • True if rename successfully, False if fail

Example

success = dataset.rename('new name')
if success:
    print('rename successfully')
